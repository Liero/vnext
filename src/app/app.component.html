Save with Blazor Server

I always strive for better productivity and quality and Blazor is a new revolutionary web framework that helps to achieve those goals.

Using Blazor Server and carefully chosen technology stack you can lower development cost of your business web application by 40% while still improving maintainability, testability, startup speed and even overall quality. How is that possible?

Single technology stack for backend and frontend is tremendous advantage, especially when it is .NET, because all the code is written in the same programming language, using the same tools and by the same developers.

Additionally:

  • It saves time spent on API's twice - on backend and frontend. No serialization, deserialization, swagger, no API authorization, less security issues.
  • Single test framework covers BE and FE and without browser automation which makes testing much easier.
  • There is no need to worry about bundle sizes which typically grow very quickly and increases traditional SPA startup time. No need to worry about lazy loading and related issues..
  • Small learning curve for existing C# developers, because they can reuse their existing skills. Blazor is just a tiny layer build on technologies well known open source and battle tested technologies like ASP.NET and Razor.
